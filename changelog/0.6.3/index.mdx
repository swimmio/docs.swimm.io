# Swimm 0.6.3 Has Been Released!

We're pleased to announce the availability of **Swimm 0.6.3**<SwimmMoji text="release"/>

<SwimmReleaseBlogPost version="0.6.3" />

<SwimmReleaseVideo version="0.6.3" />

## New Features:

 - Swimm Playlists are now DAC (docs as code) and saved in the `swm.md` format.
 - Much more intuitive and guided onboarding experience from signup to merging.

## Improvements:

 - CI Integration sample configurations updated with packed CLI binary location.
 - External links in Swimm content now open in a new tab by default.

## Bug Fixes:

 - Fixed keyboard shortcuts menu mismatch.
 - Fixed an issue in the VSCode IDE Plugin that prevented the explorer view from populating.
 - Fixed issue where `/link` in the editor wasn't scoping docs correctly without refreshing.
 - Fixed scenario where Mac command key was causing inadvertent saves while editing.

If you have any questions or issues, please reach out on our <SwimmLink target="slack" />. You can
also drop in during our next <SwimmLink target="officeHours" /> event. We hope to see you there!
