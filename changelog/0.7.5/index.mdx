# Swimm 0.7.5 Has Been Released!

We're pleased to announce the availability of **Swimm 0.7.5**<SwimmMoji text="release"/>

<SwimmReleaseBlogPost version="0.7.5" />

<SwimmReleaseVideo version="0.7.5" />

## New Features:
 - Members can directly request to join a workspace from a PR comment or Github Check, and admins will get notified.
 - See a list of recommended docs referencing the same code in the Github Check
 - Ability to add tables in your documents
 - Capability to view the published version of a doc while you're editing it


## Improvements:
 - Docs can be filtered as draft, pending, or published
 - Local draft documents are preserved even when you logged out. 


For further information and any questions, please reach out to us in our <SwimmLink target="slack" />.
