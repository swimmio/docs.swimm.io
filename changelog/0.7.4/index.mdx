# Swimm 0.7.4 Has Been Released!

We're pleased to announce the availability of **Swimm 0.7.4**<SwimmMoji text="release"/>

<SwimmReleaseBlogPost version="0.7.4" />

<SwimmReleaseVideo version="0.7.4" />

## New Features:
- Introducing GitHub App Auto-sync approval! You can now approve all auto-synced suggestions on a Pull Request with a click of a button directly from GitHub. Check out our new Pull Request comment and Checks section. 


## Improvements:
 
 - Swimm Links can optionally be opened in a new window.
 - Document drafts can now be saved on any branch.


## Bug Fixes and Deprecations:

 - Since February 9, 2022 - Quick “Login with GitHub” option is no longer available when you sign into Swimm. 

If you have any questions or issues, please reach out on our <SwimmLink target="slack" />.
